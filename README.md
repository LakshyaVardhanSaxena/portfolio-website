# 💼 Personal Portfolio Website

A modern, responsive, and visually appealing personal portfolio website built using **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**. This portfolio showcases my skills, education, experience, projects, and contact information with a clean and professional design.

The website is fully responsive and optimized for desktop, tablet, and mobile devices, providing an engaging user experience.
